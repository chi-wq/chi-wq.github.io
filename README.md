# chi-wq.github.io

chi-wq の個人 GitHub Pages サイトです。

## 📝 プロジェクト概要

このプロジェクトは、個人ウェブサイトをホストするための GitHub Pages プロジェクトです。メインページは自動的に `/pages/` サブディレクトリにリダイレクトされます。

## 🌐 アクセス先

- **メインサイト**: [https://chi-wq.github.io/](https://chi-wq.github.io/)
- **実際のページ**: [https://chi-wq.github.io/pages/](https://chi-wq.github.io/pages/)

## 📁 プロジェクト構成

```
.
├── index.html    # リダイレクトページ、/pages/ に自動転送
└── README.md     # プロジェクト説明ドキュメント
```

## 🚀 技術実装

- **ホスティングプラットフォーム**: GitHub Pages
- **リダイレクト方式**: 
  - HTML meta refresh
  - JavaScript window.location.replace
  - フォールバックリンククリック

## 📄 ライセンス

このプロジェクトは個人サイトであり、GitHub Pages の利用規約に従います。
